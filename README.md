## About Me

I'm a software engineer who builds **technical, performance-conscious software** across machine learning, graphics/engine programming, full-stack web development, and application tooling.

My work spans:

- Training and serving ML models with reproducible pipelines and API layers
- Graphics and simulation systems with low-level control in C/C++
- Full-stack web application development with Angular, ASP.NET, and relational databases
- Technical tools and desktop applications in C#
- Gameplay and engine-side development in Unreal-focused workflows

I care about software that is not just functional but **engineered well** — clear architecture, thoughtful abstractions, strong debugging surfaces, and maintainable implementation details.
